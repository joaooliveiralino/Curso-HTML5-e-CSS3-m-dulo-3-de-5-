# 💻 Estudos de HTML5 e CSS3 - Módulo 3

Repositório criado para armazenar meus exercícios e projetos desenvolvidos durante o **Módulo 3** do curso de HTML5 e CSS3 do [Curso em Vídeo](https://www.cursoemvideo.com/), ministrado pelo professor Gustavo Guanabara.

## 🚀 Sobre o Módulo
Nesta etapa, o foco foi sair do design estático e avançar para conceitos modernos de Web Design, com ênfase em:

* **Tabelas:** Criação de tabelas simples e complexas (uso de `thead`, `tbody`, `tfoot` e escopo).
* **Design Responsivo:** Conceitos de Viewport e adaptação de layout para diferentes tamanhos de tela.
* **Mobile-First:** A estratégia de desenvolver primeiro para dispositivos móveis e depois expandir para desktop.
* **Media Queries:** Uso de `@media screen` para controlar estilos baseados na largura da tela.
* **Layouts Modernos:** Práticas de design adaptável e uso de fontes externas.

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
* Google Fonts
* Ícones (Material Icons/FontAwesome)

## 📂 Projeto em Destaque
O principal desafio deste módulo foi a criação de um site totalmente responsivo (como o **Projeto Android**), onde apliquei:
1. Menus adaptáveis para mobile.
2. Imagens que se ajustam dinamicamente ao ecrã.
3. Organização de conteúdo que se reorganiza de acordo com o dispositivo.
